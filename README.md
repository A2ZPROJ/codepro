# CodePro — Gestão de Documentos

App desktop para codificação de documentos de engenharia — A2Z Projetos.

## Desenvolvimento

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Release

Crie uma tag para disparar o build automático:

```bash
git tag v1.0.0
git push origin v1.0.0
```
